### 💪🏻 Motivation
◾ 클라이언트, 백엔드, 서버 배포의 전 과정을 경험하고, 복합적인 도메인 설계를 구성하기 위해서 프로젝트를 기획하게 되었습니다.
<br>
◾ 여러 서비스의 기초가 되는 이커머스 도메인을 선택했고, 새로운 기술에 대한 경험을 위해 Spring security와 블루-그린 무중단 배포를 적용했습니다. 

<br>
